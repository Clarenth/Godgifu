# Godgifu
2024 New Technologies mega-project. Godgifu is a worldbuilding themed around the Galactic Imperium from the 4X strategy game Stellaris.

## Description
Godgifu will include a web client written in TypeScript using [Svelte](https://svelte.dev/) and [SvelteKit](https://kit.svelte.dev/), and cross-platform Desktop clients [Taur](https://tauri.app/). The web server will be written using [Rust](https://www.rust-lang.org/) with the [Tokio runtime](https://tokio.rs/) using either [Actix-Web](https://actix.rs/docs/) or [Axum](https://docs.rs/axum/latest/axum/index.html#).

## Goals
Godgifu's main goal is to be an *Overengineered Project*, meaning that it include technologies and libraries that are not necessary, but that I am interested in learning. It will prmarily feature new emerging technologies, and technologies that I have not worked with before.
